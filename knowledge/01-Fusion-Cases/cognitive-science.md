# 인지과학 (Cognitive Science)

> **"마음은 프로그램이고, 뇌는 하드웨어다."**
> — 인지혁명의 핵심 은유 (1956-)

---

## 1. 서론: 마음의 과학은 어떻게 탄생했는가

### 1.1 경이로운 현상 (The Phenomenon)

1956년 9월, MIT에서 "정보이론 심포지엄"이 열렸다. 그 자리에는 각기 다른 분야의 젊은 혁명가들이 모여 있었다:

```
│   │Chomsky  │  │ Miller  │  │ Newell  │  │McCarthy │           │
```

|-----|------------|-----------|

| 신경과학 | 반사 궁 | 신경 네트워크 |
| 인류학 | 문화 기술 | 인지 인류학 |

### 1.2 왜 이 융합이 일어났는가

**수렴의 조건들:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    인지혁명의 수렴 조건                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. 기술적 조건: 컴퓨터의 등장 (1940s-50s)                      │
│      → "계산하는 기계"가 "사고하는 기계" 가능성 시사             │
│                                                                 │
│   2. 이론적 조건: 정보이론 (Shannon, 1948)                       │
│      → 정보를 수학적으로 정량화 가능                             │
│                                                                 │
│   3. 위기 조건: 행동주의의 한계 노출                             │
│      → 언어, 기억, 추론 설명 불가                                │
│                                                                 │
│   4. 개념적 조건: "마음 = 소프트웨어" 은유                       │
│      → 뇌의 물리적 구현과 독립적인 "마음" 연구 가능              │
│                                                                 │
│   5. 제도적 조건: 학제간 연구 지원 시작                          │
│      → Sloan Foundation, 군 연구비                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
• 심리학 + 신경과학 = 인지 신경과학

...
```

## 2. 모학문 분석 (Parent Disciplines)

### 2.1 심리학 (Psychology)

**행동주의의 지배와 몰락:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    행동주의 vs 인지주의                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   행동주의 (Watson, Skinner, 1920-1950s)                        │
│   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━                           │
│   원칙: "관찰 불가능한 것은 과학의 대상이 아니다"                │
│   모델: S → R (자극 → 반응)                                     │
│   한계: 언어 습득, 문제 해결, 기억 구조 설명 불가                │
│                                                                 │
│   인지주의 (Miller, Broadbent, 1956-)                           │
│   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━                           │
│   원칙: "마음의 내부 과정을 정보처리로 모델링"                   │
│   모델: S → [정보처리] → R                                      │
│   도구: 반응시간, 오류 패턴, 프로토콜 분석                       │
│                                                                 │
│   전환점: Miller (1956) "The Magical Number Seven"              │
│   → 작업 기억의 용량 제한 = 7±2 청크                            │
│   → 내부 구조의 경험적 증거                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 언어학 (Linguistics)

**촘스키 혁명:**

```
Chomsky의 핵심 논증 (1957, 1959):

1. 언어의 창조성 (Creativity):
   "무한한 새 문장을 생성/이해할 수 있다"
   → 행동주의의 강화학습으로 설명 불가

2. 자극의 빈곤 (Poverty of Stimulus):
   "아이들이 듣는 입력만으로는 문법 학습 불가"
   → 선천적 언어 능력(LAD) 가정 필요

3. 규칙 기반 시스템:
   문법 = 유한한 규칙으로 무한 문장 생성
   → 마음 = 규칙 기반 계산 시스템

┌─────────────────────────────────────────────────────────────────┐
│                    생성문법의 핵심                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   S → NP + VP                                                   │
│   NP → (Det) + N + (PP)                                         │
│   VP → V + (NP) + (PP)                                          │
│   PP → P + NP                                                   │
│                                                                 │
│   예: "The cat sat on the mat"                                  │
│                                                                 │
│         S                                                       │
│        / \                                                      │
│      NP   VP                                                    │
│     /|\   /|\                                                   │
│   Det N  V  PP                                                  │
│   The cat sat /\                                                │
│              P  NP                                              │
│             on /\                                               │
│              Det N                                              │
│              the mat                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 인공지능 (Artificial Intelligence)

**기호주의 AI의 탄생:**

```
1956 Dartmouth Conference:
"모든 학습과 지능의 측면은 원칙적으로
기계가 시뮬레이션할 수 있을 정도로 정확히 기술될 수 있다."

Newell & Simon의 기호주의 가설:
1. 물리 기호 시스템 가설 (PSSH):
   "물리 기호 시스템은 일반 지능 행동의 필요충분조건"

2. Logic Theorist (1956):
   → 러셀-화이트헤드의 정리 증명 자동화
   → "생각하는 기계"의 첫 데모

3. GPS (General Problem Solver, 1957):
   → 수단-목표 분석
   → 인간 문제해결의 계산 모델
```

### 2.5 신경과학 (Neuroscience)

**신경 계산의 기초:**

```
McCulloch-Pitts 뉴런 (1943):
• 뉴런 = 논리 게이트
• 신경망 = 계산 장치
• "뇌는 논리 기계"

Hebb의 학습 규칙 (1949):
"함께 발화하는 뉴런은 함께 연결된다"
Δwij ∝ xi × xj

→ 신경 수준의 학습 메커니즘
→ 심리학적 연합과 신경 가소성 연결
```

```
│   언어학: 규칙 적용, 변형                                       │
```

### 3.2 구조적 동형 (Structural Isomorphism) — 기반 패턴

**컴퓨터-마음 동형:**

```
┌─────────────────────────────────────────────────────────────────┐
│              컴퓨터 ←──── 동형 ────→ 마음                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   컴퓨터 구성요소        │  마음/뇌 구성요소                     │
│   ━━━━━━━━━━━━━━━━━━━━━━━│━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━       │
│   하드웨어 (Hardware)    │  뇌 (Brain)                          │
│   소프트웨어 (Software)  │  마음 (Mind)                         │
│   입력 (Input)           │  감각 (Sensation)                    │
│   출력 (Output)          │  행동 (Behavior)                     │
│   메모리 (Memory)        │  기억 (Memory)                       │
│   CPU 처리               │  주의/인지 처리                      │
│   프로그램 버그          │  인지 오류/편향                      │
│                                                                 │
│   핵심 동형:                                                    │
│   알고리즘 ←→ 인지 과정                                         │
│   (추상적 절차가 물리적 구현과 독립)                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 가정 전복 (Assumption Subversion) — 변환 패턴

**전복된 핵심 가정들:**

| 기존 가정 (행동주의) | 전복 | 새로운 가정 (인지주의) |
|-------------------|------|---------------------|
| 마음은 관찰 불가 → 비과학적 | → | 마음을 정보처리로 모델링 가능 |
| 학습 = 연합 | → | 학습 = 규칙 습득/가설 검증 |
| 유기체는 수동적 | → | 유기체는 능동적 정보 처리자 |
| 종 간 학습 연속성 | → | 인간 언어/인지의 특수성 |

**촘스키의 스키너 비판 (1959):**

```
Skinner의 "Verbal Behavior" (1957)에 대한 Chomsky 서평:

행동주의 주장: 언어 = 자극-반응-강화
촘스키 반박:
1. "자극"이란? (같은 그림에 다양한 반응)
2. "강화"란? (내적 만족도 강화?)
3. 문법적 문장 vs 빈번한 문장

결론: "행동주의 언어 이론은
      과학적 외피를 두른 공허한 주장"

영향: 심리학 전체의 패러다임 전환 촉발
```

### 3.4 척도 도약 (Scale Jump) — 확장 패턴

**분석 수준의 다층 구조:**

```
┌─────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   수준 3: 계산 이론 (Computational Theory)                      │
│   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━                    │
│   "무엇을 계산하고 왜?"                                         │
│   예: 시각 = 2D 망막 이미지 → 3D 세계 표상                      │
│                                                                 │
│                         ↓                                       │
│                                                                 │
│   수준 2: 알고리즘/표상 (Algorithm/Representation)              │
│   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━                    │

│                                                                 │
│                         ↓                                       │
│                                                                 │
│   수준 1: 구현 (Implementation)                                 │
│   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━                    │
│   "어떤 물리적 기제로?"                                         │
│   예: 신경 회로, 시냅스, 신경전달물질                           │
│                                                                 │
│   핵심 통찰:                                                    │
│   각 수준은 독립적으로 연구 가능하지만 모두 필요                │
│   → 학제간 분업의 정당화                                        │
│                                                                 │

```

## 4. 핵심 창발적 개념 (Emergent Concepts)

### 4.1 표상 (Representation)

**정신 표상의 특성:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    표상의 핵심 특성                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. 지향성 (Intentionality): 표상은 무언가에 "대한" 것         │
│      예: "개"라는 표상 → 실제 개들을 지칭                       │
│                                                                 │
│   2. 구성성 (Compositionality): 복합 표상 = 부분 표상의 조합    │
│      예: "빨간 공" = "빨간" + "공"                               │
│                                                                 │
│   3. 체계성 (Systematicity): 관련 능력의 묶음                   │
│      "존이 메리를 사랑한다" 이해 가능하면                       │
│      "메리가 존을 사랑한다"도 이해 가능                         │
│                                                                 │
│   4. 생산성 (Productivity): 무한한 새 표상 생성 가능            │
│      유한 어휘 + 규칙 → 무한 문장                               │
│                                                                 │
│   Fodor의 주장:                                                 │
│   "사고의 언어" (Language of Thought, LOT)                      │
│   → 마음은 기호적 표상으로 계산                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 모듈성 (Modularity)

**Fodor의 모듈 이론 (1983):**

```
모듈의 특성:
1. 영역 특수성 (Domain Specificity): 특정 입력만 처리
2. 정보 캡슐화 (Informational Encapsulation): 외부 믿음의 영향 차단
3. 의무적 적용 (Obligatory Operation): 자동 실행
4. 빠른 처리 (Fast Processing)
5. 얕은 출력 (Shallow Output)
6. 고정된 신경 구조 (Fixed Neural Architecture)
7. 특징적 고장 패턴 (Characteristic Breakdown)
8. 특징적 발달 순서 (Characteristic Ontogeny)

예: 시각, 청각, 언어 처리

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│      입력 ──→ [모듈 1] ──→ ┐                                   │
│      입력 ──→ [모듈 2] ──→ ├──→ 중앙 처리 ──→ 출력            │
│      입력 ──→ [모듈 3] ──→ ┘                                   │
│                                                                 │
│      (빠르고 자동적)        (느리고 의식적)                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.4 인지 아키텍처 (Cognitive Architecture)

**주요 모델:**

```
ACT-R (Anderson, 1993):
• 선언적 기억 (사실) + 절차적 기억 (규칙)
• 모듈 구조 + 중앙 산출 시스템
• 신경 예측 가능

SOAR (Newell, 1990):
• 범용 지능 구조
• 문제 공간 탐색
• 청킹(chunking)을 통한 학습

Global Workspace Theory (Baars, 1988):
• 의식 = 정보의 전역 방송
• 무의식 모듈 → 의식적 통합
```
