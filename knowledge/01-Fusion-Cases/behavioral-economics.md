# 행동경제학 (Behavioral Economics)

> **"경제학은 합리적 인간을 연구한다. 심리학은 실제 인간을 연구한다.
> 행동경제학은 그 간극을 메운다."**
> — Richard Thaler

---

## 1. 서론: 합리성의 붕괴

### 1.1 경이로운 현상 (The Phenomenon)

경제학의 핵심 가정: **호모 에코노미쿠스 (Homo Economicus)**

```
┌─────────────────────────────────────────────────────────────────┐
│                    호모 에코노미쿠스의 특성                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. 완전한 합리성 (Perfect Rationality)                        │
│      → 모든 정보를 처리하고 최적 결정                           │
│                                                                 │
│   2. 무한한 의지력 (Unlimited Willpower)                        │
│      → 장기 이익을 위해 단기 유혹 극복                          │
│                                                                 │
│   3. 순수한 자기 이익 (Pure Self-Interest)                      │
│      → 타인의 효용은 고려 안 함                                 │
│                                                                 │
│   4. 안정적 선호 (Stable Preferences)                           │
│      → 프레이밍이나 맥락에 영향 안 받음                         │
│                                                                 │
│   결과: 아름다운 수학적 모델, 정확한 예측... 이론적으로.        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**그러나 실제 인간은...**

```
실험 1: 아시아 질병 문제 (Tversky & Kahneman, 1981)

상황: 600명이 죽을 수 있는 질병

프레임 A (이득 프레임):
• 프로그램 A: 200명 확실히 살림
• 프로그램 B: 1/3 확률로 600명 살림, 2/3 확률로 아무도 못 살림

결과: 72%가 A 선택 (위험 회피)

프레임 B (손실 프레임):
• 프로그램 C: 400명 확실히 죽음
• 프로그램 D: 1/3 확률로 아무도 안 죽음, 2/3 확률로 600명 죽음

결과: 78%가 D 선택 (위험 추구)

문제: A=C, B=D인데 선택이 다름!
→ 합리적 선택 이론 위배
```

### 1.2 융합의 결정적 순간

**1979년: 전망 이론 (Prospect Theory)**

Kahneman & Tversky의 *Econometrica* 논문 "Prospect Theory: An Analysis of Decision under Risk"

```
┌─────────────────────────────────────────────────────────────────┐
│                    전망 이론의 핵심 발견                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. 참조점 의존성 (Reference Dependence)                       │
│      효용은 절대 수준이 아닌 참조점 대비 변화로 평가            │
│                                                                 │
│   2. 손실 회피 (Loss Aversion)                                  │
│      손실의 고통 ≈ 2× 동등한 이득의 기쁨                        │
│      λ ≈ 2.25 (손실 회피 계수)                                  │
│                                                                 │
│   3. 확률 가중 (Probability Weighting)                          │
│      작은 확률: 과대평가 (복권 구매)                            │
│      큰 확률: 과소평가 (보험 기피)                              │
│                                                                 │
│   4. 감응도 체감 (Diminishing Sensitivity)                      │
│      이득/손실 영역 모두 가치 함수 곡선                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────┐
│                    기대효용이론 (Expected Utility Theory)        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   von Neumann-Morgenstern (1944):                               │
│                                                                 │

│   1. 완비성: 모든 선택지 비교 가능                              │

│   문제: 실제 인간 행동은 이 공리들을 체계적으로 위반            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 Herbert Simon의 제한된 합리성

**선구적 기여 (1950s):**

```
Simon의 "제한된 합리성" (Bounded Rationality):

문제:
• 완전한 정보 없음

영향:
```


### 3.2 경험적 이상 현상 (Empirical Anomaly) — 촉발 패턴

**경제학의 변칙 사례들:**

```
1. 주식 프리미엄 퍼즐 (Equity Premium Puzzle):
   주식 수익률 - 채권 수익률 ≈ 6% (역사적)
   기대효용으로 설명하려면 비현실적 위험 회피 필요
   → 손실 회피 + 근시안적 평가로 설명 (Benartzi & Thaler, 1995)

2. 부존 효과 (Endowment Effect):
   소유한 물건의 가치 > 같은 물건 구매 의향
   WTA > WTP (2-3배)
   → 손실 회피로 설명

3. 현상 유지 편향 (Status Quo Bias):
   디폴트 옵션에 대한 과도한 선호
   → 손실 프레이밍 + 노력 회피

4. 매몰 비용 오류 (Sunk Cost Fallacy):
   회수 불가능한 비용이 미래 결정에 영향
   → 손실 회피 + 심리적 회계
```

### 3.3 경계 개념 (Boundary Concept) — 연결 패턴

**"효용"의 재정의:**

```
┌─────────────────────────────────────────────────────────────────┐
│              "효용" 개념의 경계 역할                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   경제학의 "효용"              심리학의 "효용"                   │
│   (Decision Utility)          (Experienced Utility)             │
│   ━━━━━━━━━━━━━━━━━━━━        ━━━━━━━━━━━━━━━━━━━━━━           │
```

## 4. 핵심 창발적 개념 (Emergent Concepts)

### 4.1 전망 이론 (Prospect Theory)

**가치 함수:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    전망 이론 가치 함수                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   v(x)                                                          │
│    │                                                            │
│    │            이득 영역                                       │
│    │          /                                                 │
│    │        /  (오목: 위험 회피)                                │
│    │      /                                                     │
│    │    /                                                       │
│    │  /                                                         │
│ ───┼──────────────────────── x (변화)                          │
│    │\                        참조점                             │
│    │  \                                                         │
│    │    \                                                       │
│    │      \  손실 영역                                          │
│    │        \ (볼록: 위험 추구)                                 │
│    │          \                                                 │
│    │            \ (더 가파름: 손실 회피)                        │
│                                                                 │
│   수식:                                                         │
│   v(x) = x^α           if x ≥ 0  (α ≈ 0.88)                    │
│   v(x) = -λ(-x)^β      if x < 0  (β ≈ 0.88, λ ≈ 2.25)         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**확률 가중 함수:**

```
π(p) ≠ p  (주관적 확률 ≠ 객관적 확률)

특성:
• π(0) = 0, π(1) = 1 (확실한 결과는 정확히 가중)
• π(p) > p for small p (작은 확률 과대평가)
• π(p) < p for large p (큰 확률 과소평가)
• 불연속성: p=0과 p=ε, p=1과 p=1-ε 사이

수식 (Prelec, 1998):
π(p) = exp(-(-ln p)^α)  (α ≈ 0.65)
```

### 4.3 심리적 회계 (Mental Accounting)

**Thaler의 핵심 개념:**

```
정의: 사람들이 돈을 별도의 "계좌"로 분류하고 다르게 취급

원칙 (위반되는 경제학 원리):
1. 돈의 대체가능성 (Fungibility)
   - 같은 금액이라도 출처/용도에 따라 다르게 취급

2. 통합 vs 분리 (Integration vs Segregation)
   - 이득은 분리 선호: $50 + $25 > $75
   - 손실은 통합 선호: -$75 > -$50 - $25

예시:
• 예산 편성 (음식/오락/저축 분리)
• 매몰 비용 효과 (콘서트 티켓)
• 도박 자금 "하우스 머니"
• 연말 보너스 vs 월급
```

### 4.4 넛지 (Nudge)

**Thaler & Sunstein (2008):**

```
┌─────────────────────────────────────────────────────────────────┐
│                    넛지의 핵심 원리                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   정의:                                                         │
│   "자유로운 선택을 유지하면서 예측 가능한 방식으로               │
│    사람들의 행동을 개선하는 선택 설계의 요소"                    │
│                                                                 │
│   자유주의적 개입주의 (Libertarian Paternalism):                 │
│   • 선택의 자유 보존                                            │
│   • 그러나 디폴트/프레이밍으로 "더 나은" 선택 유도             │
│                                                                 │
│   주요 넛지 유형:                                               │
│   1. 디폴트 변경: 옵트인 → 옵트아웃                             │
│   2. 피드백 제공: 에너지 사용량 비교                            │
│   3. 사회적 규범: "대다수가 이렇게 합니다"                      │
│   4. 단순화: 복잡한 양식 → 간단한 양식                          │
│   5. 현저성: 중요 정보 강조                                     │
│                                                                 │
│   성공 사례:                                                    │
│   • 퇴직연금 자동 가입: 가입률 49% → 86%                        │
│   • 장기기증 디폴트: 옵트아웃 국가 > 옵트인 국가                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
│         ├──→ 신경경제학 (Neuroeconomics, 2000s)                 │

│         │                                                       │

│         │                                                       │
│         ├──→ 행동법경제학 (Behavioral Law & Econ)               │

```

### 5.2 정책 응용

| 정책 분야 | 행동경제학 적용 | 효과 |
|----------|---------------|------|
| **퇴직 저축** | 자동 가입 + 자동 증액 | 저축률 3배 증가 |
| **장기 기증** | 옵트아웃 디폴트 | 기증률 90%+ |
| **에너지 절약** | 사회 비교 피드백 | 2-4% 절감 |
| **세금 납부** | 사회 규범 메시지 | 체납률 감소 |
| **건강 행동** | 건강 식품 배치 | 선택 비율 증가 |

### 5.3 기업 응용

