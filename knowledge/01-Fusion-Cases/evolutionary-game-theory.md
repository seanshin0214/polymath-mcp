# 진화 게임 이론 (Evolutionary Game Theory)

> **"게임 이론은 합리적 인간을 위해 발명되었다. 그러나 박테리아도 게임을 한다."**
> — John Maynard Smith, 1982

---

## 1. 서론: 이기적 유전자의 수학

### 1.1 경이로운 현상 (The Phenomenon)

1928년, 폰 노이만은 합리적 경제 주체들의 전략적 상호작용을 분석하기 위해 게임 이론을 창안했다. 45년 후, 영국의 진화생물학자 존 메이너드 스미스는 충격적인 질문을 던졌다:

**"뇌도 없는 동물들이 어떻게 '최적 전략'을 선택하는가?"**

```
┌─────────────────────────────────────────────────────────────────┐
│                    융합의 역설                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   게임 이론의 전제              진화생물학의 관찰                │
│   ━━━━━━━━━━━━━━━━━              ━━━━━━━━━━━━━━━━━━━              │
│   • 합리적 선택자               • 본능적 행동                   │
│   • 의식적 계산                 • 무의식적 반응                  │
│   • 효용 극대화 의도            • 적응도 결과                   │
│   • 일회성 상호작용             • 세대 간 누적                   │
│                                                                 │
│                        ▼                                        │
│              ┌──────────────────┐                               │
│              │ 어떻게 양립하나? │                               │
│              └──────────────────┘                               │
│                                                                 │
│   답: "합리성"을 "적응도"로 재해석하면 동일 수학 적용 가능       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 핵심 통찰의 탄생

1973년, Maynard Smith와 George Price는 *Nature*에 "The Logic of Animal Conflict"를 발표했다. 이 논문의 핵심 혁신:

| 게임 이론 개념 | 진화적 재해석 | 새로운 의미 |
|--------------|-------------|-----------|
| 플레이어 | 유전자/전략 | 개체가 아닌 전략이 경쟁 단위 |
| 합리적 선택 | 자연선택 | "선택"은 의식이 아닌 차등 번식 |
| 내쉬 균형 | ESS | 침입 불가능한 진화적 안정 상태 |
| 효용 함수 | 적응도 | 번식 성공의 기대값 |

## 2. 모학문 분석 (Parent Disciplines)

### 2.1 게임 이론 (Game Theory)

**핵심 개념 (1944-1970):**

```
┌─────────────────────────────────────────────────────────────────┐
│                    게임 이론의 핵심 구조                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. 플레이어 집합 N = {1, 2, ..., n}                           │
│   2. 전략 공간 Si (각 플레이어의 가능한 행동들)                  │
│   3. 보수 함수 ui: S1 × S2 × ... × Sn → ℝ                      │
│                                                                 │
│   내쉬 균형 (Nash Equilibrium):                                 │
│   전략 프로필 s* = (s1*, s2*, ..., sn*) where                   │
│   ∀i, ∀si ∈ Si: ui(si*, s-i*) ≥ ui(si, s-i*)                  │
│                                                                 │
│   해석: 어떤 플레이어도 일방적 이탈로 이득을 볼 수 없는 상태     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**주요 기여자:**
- **John von Neumann** (1928): 미니맥스 정리
- **John Nash** (1950): 비협조 게임 균형
- **Thomas Schelling** (1960): 전략적 행동

**한계:**
- 완전한 합리성 가정 (현실과 괴리)
- 균형의 다중성 문제
- 동적 과정 설명 불가

### 2.2 진화생물학 (Evolutionary Biology)

**핵심 개념 (1859-1970):**

```
┌─────────────────────────────────────────────────────────────────┐
│                    진화생물학의 핵심 구조                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   자연선택의 3조건 (Lewontin, 1970):                            │
│   1. 변이 (Variation): 개체 간 형질 차이 존재                   │
│   2. 유전 (Heredity): 형질이 자손에게 전달                      │
│   3. 차등 번식 (Differential Reproduction): 적응도 차이          │
│                                                                 │
│   적응도 (Fitness) w:                                           │
│   w = (생존 확률) × (번식 성공) × (자손 생존)                   │
│                                                                 │
│   진화 동역학:                                                  │
│   Δp = p(1-p) × s  (p: 대립유전자 빈도, s: 선택계수)           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**주요 기여자:**
- **Charles Darwin** (1859): 자연선택
- **Ronald Fisher** (1930): 집단유전학 수학화
- **William Hamilton** (1964): 포괄적응도 이론

**한계:**
- 빈도 의존적 선택 설명 어려움
- 사회적 행동의 진화 설명 불완전
- 동물 갈등의 의례화 설명 불가

## 3. 융합 메커니즘: 사용된 Fusion Patterns

### 3.1 구조적 동형 (Structural Isomorphism) — 핵심 패턴

**동형 매핑:**

```
┌─────────────────────────────────────────────────────────────────┐
│              게임 이론 ←──── 동형 ────→ 진화 역학                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   게임 요소              │  진화 요소                           │
│   ━━━━━━━━━━━━━━━━━━━━━━━│━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━       │
│   플레이어 i            │  유전자/전략 타입 i                   │
│   전략 si               │  표현형 행동 패턴                     │
│   보수 ui               │  적응도 wi                            │
│   혼합 전략 σi          │  집단 내 전략 빈도 p                  │
│   기대 보수 E[ui]       │  평균 적응도 w̄                       │
│   내쉬 균형             │  진화적 안정 전략 (ESS)               │
│                                                                 │
│   핵심 변환:                                                    │
│   max ui(si, s-i) ──→ max wi(strategy, population state)       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**수학적 동형 증명:**

```
게임 이론 (복제자 역학):
ẋi = xi [fi(x) - φ(x)]

여기서:
- xi: 전략 i를 사용하는 플레이어 비율
- fi(x): 전략 i의 기대 보수
- φ(x): 집단 평균 보수

진화 역학 (Price 방정식):
Δz̄ = Cov(w, z)/w̄

변환: fi(x) → wi(x), xi → pi (유전자 빈도)

결과: 두 방정식은 수학적으로 동치
```

### 3.2 은유적 전이 (Metaphorical Transfer) — 촉발 패턴

**초기 은유적 통찰:**

메이너드 스미스의 원래 영감은 은유에서 시작되었다:

```
"동물들의 싸움을 보면서 생각했다:
 '왜 그들은 서로를 죽이지 않는가?'
 마치 '신사협정'을 지키는 것 같았다.
 그런데 신사협정은 합리적 선택을 전제하지 않나?"

 은유: 동물 갈등 = 인간의 전략적 게임

 후속 질문: 이 은유가 수학적으로 정당화될 수 있는가?
```

### 3.3 가정 전복 (Assumption Subversion) — 확장 패턴

**전복된 가정:**

| 원래 가정 (게임 이론) | 전복 | 새로운 가정 (진화 게임 이론) |
|---------------------|------|---------------------------|
| 플레이어는 합리적 | → | 전략은 유전적으로 결정됨 |
| 의식적 계산 | → | 자연선택이 "계산"을 대체 |
| 일회성 게임 | → | 반복 + 세대 간 누적 |
| 균형으로 즉시 수렴 | → | 점진적 진화 과정 |

**전복의 결과:**

```
합리성 가정 제거 후에도 게임 이론의 수학적 구조는 유지됨
⟹ 게임 이론의 "진정한" 적용 범위가 확장됨
⟹ 합리성은 게임 이론의 "필수 조건"이 아님이 증명됨
```

## 4. 핵심 창발적 개념 (Emergent Concepts)

### 4.1 진화적 안정 전략 (ESS)

**정의 (Maynard Smith, 1982):**

```
전략 I가 ESS이려면 ∀ 침입 전략 J ≠ I에 대해:

(1) E(I, I) > E(J, I)  — I는 자신에 대해 J보다 우월
                        또는
(2) E(I, I) = E(J, I) 이고 E(I, J) > E(J, J)
    — 동점일 경우 I는 J에 대해 더 잘 대응

여기서 E(A, B) = 전략 A가 전략 B를 상대할 때의 기대 보수
```

**ESS vs 내쉬 균형:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    균형 개념 비교                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   내쉬 균형 (NE)        진화적 안정 전략 (ESS)                  │
│   ━━━━━━━━━━━━━━━       ━━━━━━━━━━━━━━━━━━━━━━                  │
│                                                                 │
│   ∀ NE는 ESS인가? ─────────────────────────── NO               │
│   (내쉬 균형이 침입에 취약할 수 있음)                            │
│                                                                 │
│   ∀ ESS는 NE인가? ─────────────────────────── YES              │
│   (ESS는 항상 내쉬 균형의 부분집합)                              │
│                                                                 │
│              ┌─────────────────┐                                │
│              │  내쉬 균형 (NE) │                                │
│              │   ┌─────────┐   │                                │
│              │   │   ESS   │   │                                │
│              │   └─────────┘   │                                │
│              └─────────────────┘                                │
│                                                                 │
│   ESS = NE + 침입 안정성 조건                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 매-비둘기 게임 (Hawk-Dove Game)

**고전적 예시:**

```
보수 행렬:
             상대방
             매(H)      비둘기(D)
자신   매(H)   (V-C)/2, (V-C)/2    V, 0
      비둘기(D)   0, V            V/2, V/2

V = 자원의 가치, C = 싸움 비용

조건: C > V (싸움 비용이 자원 가치보다 큼)
```

**ESS 분석:**

```
Case 1: C > V (고비용 갈등)
  ESS = 혼합 전략: p* = V/C (매 전략 확률)

  생물학적 해석:
  집단의 V/C 비율이 공격적이고 (1-V/C)가 평화적
  → 공격성의 진화적 "적정 수준" 설명

Case 2: C < V (저비용 갈등)
  ESS = 순수 전략: 항상 매

  생물학적 해석:
  싸움이 저렴할 때 모든 개체가 공격적
```

### 4.3 복제자 역학 (Replicator Dynamics)

**기본 방정식:**

```
ẋi = xi [πi(x) - π̄(x)]

여기서:
- xi = 전략 i의 집단 내 빈도
- πi(x) = 전략 i의 현재 집단 상태에서의 적응도
- π̄(x) = Σj xj πj(x) = 집단 평균 적응도

해석:
- πi > π̄ → xi 증가 (평균보다 나은 전략은 확산)
- πi < π̄ → xi 감소 (평균보다 못한 전략은 감소)
- πi = π̄ → xi 일정 (균형 상태)
```

**Price 방정식과의 연결:**

```
George Price의 기여 (1970):
Δz̄ = Cov(w, z)/w̄ + E(wΔz)/w̄

첫 번째 항: 선택 효과 (자연선택)
두 번째 항: 전달 편향 (돌연변이, 발달)

복제자 역학은 Price 방정식의 특수 경우
(전달 편향 = 0, 연속 시간 버전)
```

### 5.3 현대적 확장

**공간 게임 (Spatial Games):**
```
• 격자 위의 진화 게임
• 국소적 상호작용 → 전역 패턴
• 협력의 공간적 유지 설명
```

**확률적 게임 (Stochastic Games):**
```
• 유한 집단 효과
• 유전적 부동 + 선택 상호작용
• Moran 과정, Wright-Fisher 모델
```

**다수준 선택 (Multi-level Selection):**
```
• 개체 vs 집단 선택
• 집단 간 경쟁의 게임 이론
• Simpson의 역설 해결
```

### 워크시트 2: 복제자 역학 시뮬레이션 가이드

```python
# 복제자 역학 시뮬레이션 템플릿

import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

def replicator_dynamics(x, t, payoff_matrix):
    """
    복제자 역학 미분방정식

    Parameters:
    - x: 전략 빈도 벡터 [x1, x2, ..., xn]
    - t: 시간
    - payoff_matrix: n×n 보수 행렬

    Returns:
    - dx/dt: 빈도 변화율
    """
    # 각 전략의 적응도
    fitness = payoff_matrix @ x

    # 평균 적응도
    avg_fitness = x @ fitness

    # 복제자 방정식
    dxdt = x * (fitness - avg_fitness)

    return dxdt

# 예시: 매-비둘기 게임
V, C = 2, 4  # 자원 가치, 싸움 비용
payoff = np.array([
    [(V-C)/2, V],    # 매의 보수
    [0, V/2]          # 비둘기의 보수
])

# 초기 조건
x0 = [0.9, 0.1]  # 90% 매, 10% 비둘기

# 시뮬레이션
t = np.linspace(0, 50, 1000)
solution = odeint(replicator_dynamics, x0, t, args=(payoff,))

# 이론적 ESS: p* = V/C = 0.5
print(f"이론적 ESS (매 비율): {V/C}")
print(f"시뮬레이션 최종 상태: {solution[-1]}")
```
