# 계산신경과학 (Computational Neuroscience)

> **"뇌를 이해하려면, 뇌가 무엇을 계산하는지 물어야 한다."**
> — David Marr, 1982

---

## 1. 서론: 뇌의 알고리즘

### 1.1 경이로운 현상 (The Phenomenon)

1952년, 영국의 두 생리학자 Alan Hodgkin과 Andrew Huxley는 역사적인 발견을 했다:

```
┌─────────────────────────────────────────────────────────────────┐
│                    Hodgkin-Huxley의 혁명 (1952)                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   질문: 신경세포는 어떻게 전기 신호를 만드는가?                  │
│                                                                 │
│   방법:                                                         │
│   • 오징어 거대 축삭 (직경 ~1mm)                                │
│   • 전압 클램프 실험                                            │
│   • 정밀한 전류 측정                                            │
│                                                                 │
│   발견:                                                         │
│   • Na⁺, K⁺ 이온 채널의 전압 의존적 개폐                        │
│   • 활동전위의 발생 메커니즘                                    │
│                                                                 │
│   혁신:                                                         │
│   생물학적 현상을 미분방정식으로 완벽히 기술!                   │
│                                                                 │
│   C(dV/dt) = -ḡ_Na m³h(V-E_Na) - ḡ_K n⁴(V-E_K) - ḡ_L(V-E_L) + I │
│                                                                 │
│   노벨 생리의학상 (1963)                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 왜 이 융합이 필요했는가

**뇌의 복잡성 문제:**

```
인간 뇌의 규모:
• ~86억 개 뉴런
• ~150조 개 시냅스
• 각 뉴런: ~7,000개 시냅스 연결
• 신호 처리: 밀리초 단위

전통적 접근의 한계:
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│   생리학만으로는...        심리학만으로는...                    │
│   ━━━━━━━━━━━━━━━━━━        ━━━━━━━━━━━━━━━━━━                  │
│   • 뉴런 하나는 이해       • 행동은 기술                        │
│   • 전체 시스템? 불가능    • 메커니즘? 블랙박스                │
│                                                                │
│                     ↓ 필요한 것 ↓                              │
│                                                                │
│              ┌──────────────────────┐                          │
│              │  수학적/계산적 모델  │                          │
│              │  (Computational      │                          │
│              │   Models)            │                          │
│              └──────────────────────┘                          │
│                                                                │
│   부분 → 전체로 가는 원칙적 방법                               │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 1.3 이 융합이 특별한 이유

**다중 수준의 통합:**

```
계산신경과학의 독특한 위치:

         철학/인지과학
              │
              ▼
┌─────────────────────────────┐
│     심리학/행동과학         │  ← "무엇을" (What)
├─────────────────────────────┤

│   (Computational            │     알고리즘/표상

├─────────────────────────────┤
│     세포/분자 신경과학      │  ← "무엇으로" (With What)

              │

## 2. 모학문 분석 (Parent Disciplines)

### 2.1 신경생리학 (Neurophysiology)

**실험적 기반:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    신경생리학의 핵심 발견                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. 활동전위 (Action Potential)                                │
│      • 전부-아니면-전무 (All-or-None)                           │
│      • 약 1ms 지속, -70mV → +40mV → -70mV                      │
│      • 축삭을 따라 전파                                         │
│                                                                 │
│   2. 시냅스 전달 (Synaptic Transmission)                        │
│      • 화학적 시냅스: 신경전달물질                              │
│      • 전기적 시냅스: 간극연접                                  │
│      • 흥분성 (EPSP) vs 억제성 (IPSP)                          │
│                                                                 │
│   3. 시냅스 가소성 (Synaptic Plasticity)                        │
│      • LTP (장기강화): 학습의 세포적 기반                       │
│      • LTD (장기약화): 망각/가지치기                            │
│      • Hebb 규칙: "Fire together, wire together"               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 수학/물리학 (Mathematics/Physics)

**이론적 도구:**

```
핵심 수학 도구:

1. 미분방정식:
   • Hodgkin-Huxley 방정식
   • 케이블 방정식 (수상돌기)
   • 발화율 모델

2. 동역학 시스템:
   • 안정점, 극한 주기
   • 분기(Bifurcation) 분석
   • 위상평면 분석

3. 확률론/통계역학:
   • 포아송 과정 (스파이크 트레인)
   • 정보이론 (신경 코딩)
   • 최대 엔트로피 모델

4. 선형대수/최적화:
   • 신경망의 가중치 행렬
   • 경사하강법
   • 베이즈 추론
```

### 2.3 컴퓨터과학/AI (Computer Science/AI)

**계산적 사고:**

```
컴퓨터과학의 기여:

1. 알고리즘적 관점:
   "뇌가 해결하는 문제를 컴퓨터가 어떻게 풀 것인가?"
   → 역으로 뇌의 알고리즘 추론

2. 신경망 모델:
   • 퍼셉트론 (Rosenblatt, 1958)
   • 역전파 (Rumelhart et al., 1986)
   • 딥러닝 (현재)

3. 시뮬레이션:
   • 대규모 신경망 시뮬레이션
   • NEURON, NEST, Brian

4. 기계학습과의 상호영향:
   뇌 → AI: 영감
   AI → 뇌: 가설/예측
```

```

           ┌───────────────┐
           │   신경 코드   │
           │ (Neural Code) │
           └───────────────┘
                  │
    ┌─────────────┼─────────────┐
    │             │             │
    ▼             ▼             ▼
 발화율 코드   시간 코드    집단 코드
 (Rate Code)  (Temporal)  (Population)

핵심 질문들:
• 정보가 어떻게 표상되는가?
• 효율은 어느 정도인가?
• 어떻게 해독(decode)하는가?
```

## 4. 핵심 창발적 개념 (Emergent Concepts)

### 4.1 Hodgkin-Huxley 모델

**활동전위의 수학:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    Hodgkin-Huxley 방정식                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   C_m (dV/dt) = -I_Na - I_K - I_L + I_ext                      │
│                                                                 │
│   I_Na = ḡ_Na · m³ · h · (V - E_Na)  (나트륨 전류)             │
│   I_K  = ḡ_K · n⁴ · (V - E_K)        (칼륨 전류)               │
│   I_L  = ḡ_L · (V - E_L)              (누출 전류)               │
│                                                                 │
│   게이팅 변수 역학:                                             │
│   dm/dt = α_m(V)(1-m) - β_m(V)m                                │
│   dh/dt = α_h(V)(1-h) - β_h(V)h                                │
│   dn/dt = α_n(V)(1-n) - β_n(V)n                                │
│                                                                 │
│   α, β = 전압 의존적 열림/닫힘 속도                             │
│                                                                 │
│   의의:                                                         │
│   • 생물학적 현상의 완벽한 수학적 기술                          │
│   • 계산신경과학의 출발점                                       │
│   • 이온 채널 연구의 이론적 기반                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 집단 코딩 (Population Coding)

**뉴런 집단의 정보 표상:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    집단 코딩 개념                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   개별 뉴런:                                                    │
│   • 잡음이 많음                                                 │
│   • 튜닝이 넓음                                                 │
│   • 애매한 정보                                                 │
│                                                                 │
│   뉴런 집단:                                                    │
│   • 잡음 평균화                                                 │
│   • 정밀한 표상                                                 │
│   • 풍부한 정보                                                 │
│                                                                 │
│   예: 운동 방향 코딩 (Georgopoulos et al., 1986)                │
│                                                                 │
│         뉴런 1       뉴런 2       뉴런 3                        │
│   선호방향: ↑         ↗          →                             │
│   발화율:   20        50          30        (Hz)                │
│                                                                 │
│   집단 벡터:                                                    │
│   r⃗ = Σᵢ fᵢ · p̂ᵢ  (가중 합)                                   │
│                                                                 │
│   → 단일 뉴런보다 훨씬 정확한 방향 추정                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.3 예측 코딩 (Predictive Coding)

**뇌의 베이즈 추론:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    예측 코딩 프레임워크                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   핵심 아이디어:                                                │
│   "뇌는 세상의 모델을 가지고 있고,                              │
│    감각 입력과 예측의 차이(오차)만 전달한다"                    │
│                                                                 │
│   계층적 구조:                                                  │
│                                                                 │
│     고수준 (추상적 표상)                                        │
│            │                                                    │
│      예측 ↓   ↑ 오차                                           │
│            │                                                    │
│     중간 수준                                                   │
│            │                                                    │
│      예측 ↓   ↑ 오차                                           │
│            │                                                    │
│     저수준 (감각 입력)                                          │
│                                                                 │
│   수학적 형식화:                                                │
│   ε = x - f(θ)     (오차 = 입력 - 예측)                        │

│                                                                 │
│   신경 구현:                                                    │
│   • 피라미드 세포: 예측 전달                                    │
│   • 억제 뉴런: 오차 계산                                        │
│   • 조절 신호: 정밀도(precision) 가중                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

│                                                                 │
│   응용:                                                         │

│                                                                 │

```
