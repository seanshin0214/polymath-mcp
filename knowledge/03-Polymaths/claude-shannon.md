## 2. 핵심 업적

### 2.1 정보이론 (Information Theory)

```
"통신의 수학적 이론" (1948):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"20세기 가장 중요한 석사 논문 다음의 가장 중요한 논문"
   — 후대 평가

핵심 개념:

비트 (Bit):
• Binary digit의 약자
• 정보의 기본 단위
• 0 또는 1의 선택

정보 엔트로피:
H(X) = -Σ p(x) log₂ p(x)

• 메시지의 평균 정보량
• 불확실성의 척도
• 최적 압축의 하한

채널 용량 (Channel Capacity):
C = max_{p(x)} I(X;Y)

• 오류 없는 최대 전송률
• 노이즈 채널의 근본 한계
• Shannon 한계

채널 코딩 정리:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"R < C이면, 오류확률을 임의로 작게 할 수 있다"

혁명적 함의:
• 노이즈를 이길 수 있다
• 충분히 좋은 코드가 존재
• 디지털 통신의 이론적 기반

소스 코딩 정리:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"N개 심볼 → N×H(X) 비트로 압축 가능"

• 데이터 압축의 이론적 한계
• 무손실 압축: ZIP, PNG
• 손실 압축: JPEG, MP3
```

### 2.3 암호학

```
"통신의 비밀 시스템" (1945, 기밀해제 1949):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

완전 비밀 (Perfect Secrecy):
H(M|C) = H(M)

암호문을 알아도 메시지에 대한 불확실성이
변하지 않음

One-Time Pad의 증명:
• 완전 비밀의 유일한 방법
• 키 길이 ≥ 메시지 길이
• 수학적 증명 제공

혼돈과 확산 (Confusion & Diffusion):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

혼돈 (Confusion):
• 키와 암호문 관계 복잡화
• 치환 (substitution)

확산 (Diffusion):
• 평문 통계 분산
• 순열 (permutation)

현대 블록 암호 (AES, DES)의 기본 원리

2차 대전 중 암호 해독:
• 벨 연구소 X-시스템 연구
• 앨런 튜링과 협력
• 처칠-루스벨트 통신 보안
```

```
E (Embodied): 체화된 지식

• 확률론적 사고
• 추상화 능력

I (Integration): 영역 통합
• 수학 + 공학
• 물리 + 논리
• 놀이 + 연구

T (Transfer): 지식 전이
• 논리학 → 회로
• 확률론 → 통신
• 게임 → AI
```

### 5.2 성공 요인

```
1. 지적 놀이터
   ├─ Bell Labs의 자유로운 환경
   ├─ 실용적 압박 없는 연구
   └─ 호기심 중심 탐구

2. 이중 학위
   ├─ 전자공학 + 수학 학사
   ├─ 이론과 실제의 결합
   └─ 두 언어 모두 유창

3. 타이밍
   ├─ 통신 기술 폭발기
   ├─ 컴퓨터 태동기
   └─ 암호학 수요 (2차대전)

4. 시각화 능력
   ├─ 복잡한 것을 간단한 그림으로
   ├─ 통신 모델 다이어그램
   └─ 회로 도식화

5. 낮은 출판 압박
   ├─ 완성된 것만 출판
   ├─ 양보다 질
   └─ 충분히 이해될 때까지 보류
```
