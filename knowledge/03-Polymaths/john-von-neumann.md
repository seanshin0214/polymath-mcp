## 5. 폴리매시 패턴 분석

### 5.1 융합 패턴: LEAP

```
폰 노이만의 LEAP 패턴:
━━━━━━━━━━━━━━━━━━━━━━━━

L (Lateral): 측면 이동
• 수학 → 물리학 → 경제학 → 컴퓨터
• 필요에 따른 분야 전환
• 문제 중심 접근

E (Expertise): 전문성 획득
• 각 분야에서 최고 수준 달성
• 단순 관심이 아닌 기여
• 빠른 학습 능력

A (Abstraction): 추상화
• 구체적 문제 → 일반 이론
• 수학적 형식화
• 공리적 방법론

P (Projection): 적용
• 추상 이론 → 실제 문제
• 양자역학 → 핵무기
• 게임이론 → 전략
```

   ├─ 양자역학 형성기
   ├─ 컴퓨터 탄생기
   └─ 냉전 시대 과학 투자

4. 인적 네트워크
   ├─ 힐베르트, 괴델과 교류
