## 2. 핵심 업적

### 2.1 수학

```
미적분학 (독립 발명):
━━━━━━━━━━━━━━━━━━━━━━━━

뉴턴과 독립적으로 발명 (1675-1684):
• dy/dx 표기법 (현재까지 사용)
• ∫ 적분 기호 (summa의 S)
• 미분과 적분의 역관계
• 라이프니츠 규칙: d(uv) = udv + vdu

"나는 이 문제의 열쇠를 찾았다"
   — 1675년 11월 11일 노트

이진법 (Binary System):
━━━━━━━━━━━━━━━━━━━━━━━━

• 0과 1만으로 모든 수 표현
• 주역의 음양과 연결 시도
• 논리 연산의 기반
• 현대 컴퓨터의 핵심

"계산의 새로운 과학의 경이로운 기원"
   — De Progressione Dyadica (1679)

기타 수학 업적:
• 행렬식 (determinant) 발명
• 위상수학 기초 (Analysis situs)
• 무한소 해석학 체계화
• 곱의 법칙, 연쇄 법칙
```

### 2.2 철학

```
모나드론 (Monadology, 1714):
━━━━━━━━━━━━━━━━━━━━━━━━━━━

모나드 = 실체의 궁극 단위
• 비물질적, 분할 불가
• "창이 없다" (외부 영향 없음)
• 각자 내적 원리로 변화
• 사전 확립된 조화 (pre-established harmony)

"모나드는 자연의 진정한 원자다"

형이상학 원리:
━━━━━━━━━━━━━━━━━━━━━━━━━━━

충족이유율 (Principle of Sufficient Reason):
"아무것도 이유 없이 일어나지 않는다"
∀x ∃y (Reason(y, x))

식별불가자 동일성 (Identity of Indiscernibles):
"구별할 수 없는 두 존재는 동일하다"
∀P(Px ↔ Py) → x = y

연속성 원리 (Principle of Continuity):
"자연은 도약하지 않는다"

최선의 세계 (Best of All Possible Worlds):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

신정론 (Theodicy, 1710):
• 악의 문제에 대한 해답
• 신은 가능한 세계 중 최선을 선택
• 악 = 전체 선의 필요조건
• 후에 볼테르의 캉디드에서 풍자
```

### 2.3 논리학 & 계산

```
Characteristica Universalis (보편 문자):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

목표:
• 모든 개념을 기호로 표현
• 언어적 모호성 제거
• 과학적 소통의 완전성

"적절한 문자가 있다면 오류는
 계산 실수로 환원될 것이다"

Calculus Ratiocinator (추론 계산기):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 논리적 추론의 기계화
• 기호 조작으로 진리 발견
• 분쟁 해결: "계산해 봅시다!"
• 현대 컴퓨터 과학의 선구

Ars Combinatoria (조합술, 1666):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 기본 개념들의 조합으로 복잡 개념 생성
• 알파벳 of 사고
• 룰루스의 영향
• 현대 의미론적 프리미티브 개념
```

### 2.4 물리학 & 과학

```
역학 (Mechanics):
━━━━━━━━━━━━━━━━━━━━━━━━

활력 논쟁 (Vis Viva Controversy):
• 뉴턴: 운동량 mv 보존
• 라이프니츠: "활력" mv² 보존
• 결과: 둘 다 옳음 (운동량 vs 운동에너지)

"나는 힘을 다른 방식으로 측정한다"

역학의 형이상학:
• 힘 = 모나드의 물리적 발현
• 관계주의적 시공간 (vs 뉴턴의 절대시공간)
• 연속체 가설

기타 과학:
━━━━━━━━━━━━━━━━━━━━━━━━

지질학:
• 지구의 냉각 역사
• 프로토게아 (Protogaea)

광물학/고생물학:
• 화석 수집 및 분류
• 하르츠 광산 연구
```
