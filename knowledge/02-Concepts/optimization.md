# 최적화 (Optimization)

> **"자연은 극값을 좋아한다."**
> — 최소작용의 원리에서

---

## 1. 개념 개요

### 1.1 핵심 정의

```
┌─────────────────────────────────────────────────────────────────┐
│                    최적화 문제의 일반 형태                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   min   f(x)        (목적함수 최소화)                          │
│    x                                                            │
│                                                                 │
│   subject to:                                                   │
│     g_i(x) ≤ 0,  i = 1,...,m    (부등식 제약)                  │
│     h_j(x) = 0,  j = 1,...,p    (등식 제약)                    │
│     x ∈ X                        (변수 범위)                    │
│                                                                 │
│   핵심 요소:                                                    │
│   • 결정 변수 x: 우리가 선택하는 것                             │
│   • 목적함수 f: 최소화(또는 최대화)하고 싶은 것                 │
│   • 제약조건: 반드시 만족해야 하는 조건                         │
│                                                                 │
│   해의 유형:                                                    │
│   • 전역 최적해: 전체 영역에서 가장 좋은 해                     │
│   • 국소 최적해: 인근에서만 가장 좋은 해                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 왜 중요한가

- 의사결정의 수학적 프레임워크
- 자연법칙의 변분 원리
- 진화와 적응의 메커니즘
- 기계학습의 핵심 (손실 최소화)

## 2. 분야별 발현

### 2.1 수학/물리학

```
변분법 (Calculus of Variations):
"함수를 변수로 하는 최적화"

δ∫L(x, ẋ, t)dt = 0

오일러-라그랑주 방정식:
∂L/∂x - d/dt(∂L/∂ẋ) = 0

최소작용의 원리:
• 빛: 페르마 원리 (최소 시간)
• 역학: 해밀턴 원리 (최소 작용)
• 자연의 "최적화" 경향
```

### 2.2 경제학

```
효용 최대화:
max U(x₁, x₂, ..., xₙ)
s.t. Σpᵢxᵢ ≤ I  (예산 제약)

이윤 최대화:
max π = pQ - C(Q)

사회적 최적:
• 파레토 최적: 누군가를 개선하면 다른 누군가가 악화
• 후생 경제학 정리
• 시장 균형과 최적의 관계
```

### 2.3 생물학

```
적응도 최대화:
진화 = "적응도 함수의 최적화"

적응도 지형 (Fitness Landscape):
• 유전형 공간의 각 점에 적응도 할당
• 자연선택 = 지형 위의 상승
• 국소 최적의 문제 (Adaptive Peak)

최적 행동 이론:
• 최적 먹이 찾기 (Optimal Foraging)
• 최적 생활사 전략
• 행동생태학의 기반
```

### 2.4 기계학습

```
손실 최소화:
min L(θ) = Σᵢ ℓ(f(xᵢ; θ), yᵢ)

경사하강법:
θ_{t+1} = θ_t - η∇L(θ_t)

변형:
• SGD: 미니배치로 근사
• Adam: 적응적 학습률
• 2차 방법: 곡률 정보 사용

과적합과 정규화:
min L(θ) + λR(θ)
```

## 3. 최적화 방법론

### 3.1 볼록 최적화 (Convex Optimization)

```
┌─────────────────────────────────────────────────────────────────┐
│                    볼록 문제의 특성                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   볼록 함수:                                                    │
│   f(αx + (1-α)y) ≤ αf(x) + (1-α)f(y), ∀α∈[0,1]                │
│                                                                 │
│   핵심 정리:                                                    │
│   볼록 문제에서 국소 최적해 = 전역 최적해                       │
│                                                                 │
│   예:                                                           │
│   • 선형 계획법 (LP)                                            │
│   • 이차 계획법 (QP)                                            │
│   • 세미정부호 계획법 (SDP)                                     │
│                                                                 │
│   알고리즘:                                                     │
│   • 내점법 (Interior Point)                                     │
│   • 근접점법 (Proximal)                                         │
│   • ADMM                                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 비볼록 최적화

```
도전:
• 다수의 국소 최적해
• 전역 최적 보장 어려움
• NP-hard 문제 많음

접근법:
• 메타휴리스틱: 유전 알고리즘, 모의 담금질
• 다시작 (Multi-start)
• 완화 (Relaxation) 후 반올림
• 딥러닝: 경험적으로 잘 작동
```

### 3.3 다목적 최적화

```
여러 목적함수가 충돌할 때:
min [f₁(x), f₂(x), ..., fₖ(x)]

파레토 최적:
어떤 목적도 악화시키지 않고는
다른 목적을 개선할 수 없음

파레토 전선 (Pareto Front):
모든 파레토 최적해의 집합

접근법:
• 가중합: min Σwᵢfᵢ(x)
• ε-제약: min f₁ s.t. fᵢ ≤ εᵢ
• 진화 알고리즘: NSGA-II
```

## 4. 핵심 통찰

### 4.1 최적화 vs 적응

```
최적화 (공학적):
• 목적함수가 명시적으로 정의됨
• 전역 최적을 추구
• 계산적 방법 사용

적응 (자연적):
• 적응도 함수가 암묵적
• 국소 탐색 (자연선택)
• 전역 최적 보장 없음

진화는 "근시안적 최적화"
• 현재 변이 중 최선 선택
• 역사적 경로에 제약
• "좋은 enough" 해
```

### 4.2 최적성 조건

```
1차 필요조건 (무제약):
∇f(x*) = 0

2차 충분조건:
∇²f(x*) ≻ 0 (양정치)

KKT 조건 (제약 있는 경우):
• 정상성: ∇f + Σλᵢ∇gᵢ + Σμⱼ∇hⱼ = 0
• 원시 실행가능성: g(x) ≤ 0, h(x) = 0
• 쌍대 실행가능성: λ ≥ 0
• 상보 이완: λᵢgᵢ(x) = 0
```

## 6. 성찰적 질문

1. **목적함수의 선택**: 무엇을 최적화할지 어떻게 정하는가? 이것 자체가 가치 판단인가?

2. **전역 vs 국소**: 국소 최적에 갇히지 않으려면? 진화는 어떻게 이 문제를 다루는가?

3. **다목적의 딜레마**: 상충하는 목적들 사이에서 어떻게 선택하는가?

4. **최적화의 부작용**: Goodhart의 법칙 - "측정이 목표가 되면 좋은 측정이 아니다"

5. **자연과 최적**: 자연은 정말로 "최적화"하는가, 아니면 우리가 그렇게 해석하는 것인가?
