# 창발 (Emergence)

> **"전체는 부분의 합보다 크다."**
> — 아리스토텔레스 (재해석)

> **"More is different."**
> — Philip Anderson, 1972

---

## 1. 개념 개요

### 1.1 핵심 정의

```
┌─────────────────────────────────────────────────────────────────┐
│                    창발의 정의                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   기본 정의:                                                    │
│   부분들의 상호작용에서 발생하는,                               │
│   부분만으로는 예측/설명할 수 없는 전체의 속성                  │
│                                                                 │
│   약한 창발 (Weak Emergence):                                   │
│   원칙적으로는 환원 가능하나 실제로는 복잡                      │
│   예: 물의 액체 상태 (분자 역학에서 도출 가능)                  │
│                                                                 │
│   강한 창발 (Strong Emergence):                                 │
│   원칙적으로도 환원 불가능, 새로운 인과력 발생                  │
│   예: 의식? (논쟁적)                                            │
│                                                                 │
│   형식적 정의 (시도):                                           │
│   P는 시스템 S에서 창발적 ⟺                                    │
│   1. P는 S의 전역적 속성                                        │
│   2. P는 S의 부분들에는 없음                                    │
│   3. P는 부분들의 상호작용에서 발생                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 왜 중요한가

- 환원주의의 한계 인식
- 복잡계 과학의 핵심 개념
- 수준 간 관계 이해
- 새로움의 존재론적 가능성

## 2. 분야별 발현

### 2.1 물리학

```
Anderson의 "More is Different" (1972):

대칭성 깨짐 (Symmetry Breaking):
• 미시 법칙의 대칭성 → 거시 상태의 비대칭
• 예: 강자성, 초전도, 결정 구조

상전이 (Phase Transition):
• 임계점에서 새로운 속성 출현
• 창발적 질서 변수 (order parameter)
• 보편성 (universality)

"기본 입자 물리학이 고체물리학을 '환원'하지 않는다.
 새로운 수준마다 새로운 법칙이 필요하다."
```

### 2.2 생물학

```
생명의 창발적 속성:
• 세포: 분자 → 자기복제 시스템
• 발생: 단일 세포 → 복잡한 유기체
• 진화: 개체 선택 → 종의 적응
• 생태계: 종들 → 자기조절 시스템

예: 개미 군집 (Ant Colony)
• 개별 개미: 단순한 규칙 따름
• 군집 전체: 복잡한 행동 출현
  - 최적 먹이 경로
  - 온도 조절
  - 역할 분담
• "집단 지성" (Swarm Intelligence)
```

### 2.3 신경과학/의식

```
의식은 창발적인가?

뉴런 수준:
• 전기화학적 신호
• 수십억 개의 뉴런

시스템 수준:
• 통합된 경험 (Integrated Experience)
• 주관적 질감 (Qualia)
• 자아 인식 (Self-awareness)

"어려운 문제" (Hard Problem):
물리적 과정 → 주관적 경험?
이 간극은 창발로 메워지는가?
```

### 2.4 사회과학

```
사회적 창발:

거시사회학:
• 개인 행동 → 사회 구조
• 규범, 제도, 문화

경제학:
• 개별 거래 → 시장 가격
• "보이지 않는 손"
• 시장 실패와 비효율도 창발

언어:
• 개인 발화 → 문법 구조
• 언어 변화의 자발적 패턴
```

### 3.1 자기조직화 (Self-Organization)

```
┌─────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────┤

│   외부 명령 없이 시스템 내부에서 질서 발생                      │
│                                                                 │
│   조건:                                                         │
│   • 다수의 상호작용하는 요소                                    │

│   • 피드백 (양성/음성)                                          │

└─────────────────────────────────────────────────────────────────┘
```

### 3.2 비선형성과 임계성

```
비선형 상호작용:
• 원인과 결과의 비례성 깨짐
• 작은 변화 → 큰 결과 (나비 효과)
• 상전이에서 새로운 상 출현

임계성 (Criticality):
• 임계점 근처에서 창발 현상 극대화
• 멱법칙 분포
• 장거리 상관관계
• "Edge of Chaos"
```

## 4. 철학적 논쟁

### 4.1 환원주의 vs 창발주의

```
환원주의 (Reductionism):
"모든 것은 원칙적으로 물리학으로 설명 가능"
• 창발 = 인식론적 한계 (우리 능력의 문제)
• 새로운 인과력은 없음

창발주의 (Emergentism):
"새로운 수준에서 새로운 인과력 발생"
• 창발 = 존재론적 실재
• 하향 인과 (downward causation)

현대적 절충:
• 약한 창발은 대부분 수용
• 강한 창발은 논쟁 중
• 다중 실현 (multiple realization) 개념
```

### 4.2 하향 인과 (Downward Causation)

```
질문: 전체가 부분에 영향을 미칠 수 있는가?

찬성:
• 유기체가 세포를 조절
• 마음이 뇌를 바꿈 (가소성)
• 사회 구조가 개인 행동 제약

반론:
• 논리적 순환? (전체 = 부분의 관계)
• 물리적 인과 폐쇄성 위배?
• 설명적 편의 vs 실제 인과?
```

## 6. 성찰적 질문

1. **존재론적 지위**: 창발적 속성은 "진짜로" 존재하는가, 아니면 설명의 편의인가?

2. **예측 가능성**: 창발을 예측할 수 있는가? 할 수 없다면 과학적 개념인가?

3. **의식의 문제**: 의식이 강한 창발이라면, 어떻게 연구할 수 있는가?

4. **AI와 창발**: 인공 신경망에서 창발하는 것은 무엇인가? 이해인가?

5. **윤리적 함의**: 창발적 실체 (국가, 기업, AI)에 도덕적 지위를 부여해야 하는가?
