# 피드백 (Feedback)

> **"피드백은 목적 지향적 행동의 핵심 메커니즘이다."**
> — Norbert Wiener, Cybernetics (1948)

---

### 1.2 피드백의 두 유형

```
음성 피드백 (Negative Feedback):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 출력이 입력을 감소시킴
• 안정화 효과
• 목표값으로 수렴
• 예: 온도조절기, 혈당 조절

양성 피드백 (Positive Feedback):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 출력이 입력을 증가시킴
• 증폭/불안정화 효과
• 폭주 또는 상전이
• 예: 마이크 하울링, 분만 진통
```

## 2. 분야별 발현

### 2.1 제어 이론 (Control Theory)

```
PID 제어기:
u(t) = Kp·e(t) + Ki·∫e(τ)dτ + Kd·de/dt

e(t) = 목표값 - 현재값 (오차)

• P (비례): 현재 오차에 반응
• I (적분): 누적 오차 보정
• D (미분): 변화율 예측

안정성 조건:
피드백 루프의 위상과 이득이
불안정 영역을 피해야 함
```

### 2.2 생물학

```
항상성 (Homeostasis):
"내부 환경의 안정성 유지"

예: 체온 조절
┌─────────────────────────────────────────┐
│  감지: 온도 수용체                       │
│    ↓                                    │
│  비교: 시상하부 (37°C 기준)              │
│    ↓                                    │
│  반응: 떨림, 발한, 혈관 수축/확장        │
│    ↓                                    │
│  결과: 체온 변화 → 감지 (루프 완성)      │
└─────────────────────────────────────────┘

다른 예:
• 혈당 (인슐린/글루카곤)
• 혈압 (심박수, 혈관)
• 호르몬 축 (HPA, HPT, HPG)
```

### 2.3 경제학

```
시장 피드백:
• 가격 상승 → 수요 감소 (음성)
• 가격 상승 → 투기적 구매 (양성, 버블)

경기 순환:
• 승수 효과 (양성)
• 자동 안정화 장치 (음성)

금융 시스템:
• 레버리지 감소 → 자산 매도 → 가격 하락 →
  → 추가 레버리지 감소 (양성, 위기)
```

### 2.4 심리학/사회

```
인지 피드백:
• 자기효능감 → 노력 → 성공 → 자기효능감 (양성)
• 불안 → 회피 → 일시적 안도 → 불안 유지 (음성 강화)

사회적 피드백:
• 사회적 증거: 행동 → 모방 → 더 많은 행동
• 자기실현적 예언: 믿음 → 행동 → 결과 → 믿음 확인
```

## 3. 피드백 분석

### 3.1 루프 분석

```
인과 루프 다이어그램 (CLD):
A ──(+)──► B ──(-)──► C ──(+)──► A

루프 극성 = 음성 기호의 개수
• 짝수개 음성 → 양성 루프 (Reinforcing)
• 홀수개 음성 → 음성 루프 (Balancing)

시스템 역학:
dx/dt = f(x, 피드백)

선형 시스템:
dx/dt = Ax

안정성: 행렬 A의 고유값 분석
```

### 3.2 지연과 진동

```
시간 지연 (Time Delay):
피드백 신호가 즉각적이지 않음

결과:
• 과조정 (Overshoot)
• 진동 (Oscillation)
• 잠재적 불안정

예:
• 샤워 온도 조절 (파이프 지연)
• 경제 정책 효과 (시차)
• 기후 변화 대응 (관성)
```

## 4. 핵심 통찰

### 4.1 피드백과 목적론

```
사이버네틱스의 핵심 통찰:
"목적 지향적 행동 = 음성 피드백"

목표 추구 = 오차 감소 메커니즘
• 명시적 의도 필요 없음
• 메커니즘이 "목적"처럼 보이게 함
• 기계와 생명의 통합적 이해

함의:
• 생기론(vitalism) 없이 생명 현상 설명
• 인공지능의 개념적 기반
• 텔레오노미(teleonomy) 개념
```

### 4.2 양성 피드백의 역할

```
불안정성만은 아님:

창조적 역할:
• 상전이: 새로운 질서 출현
• 진화: 성공적 변이의 증폭
• 학습: 강화 메커니즘

한계 조건:
• 대부분의 양성 피드백은 한계에서 멈춤
• 자원 고갈, 공간 제약, 다른 음성 루프
• "S자 곡선" (Logistic curve)
```
