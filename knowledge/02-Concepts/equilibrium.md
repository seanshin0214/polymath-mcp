# 평형 (Equilibrium)

> **"평형은 변화가 멈추는 곳이 아니라, 변화들이 균형을 이루는 곳이다."**
> — 동적 평형의 핵심 통찰

---

## 1. 개념 개요

### 1.1 핵심 정의

```
┌─────────────────────────────────────────────────────────────────┐
│                    평형의 다양한 유형                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   정적 평형 (Static Equilibrium):                               │
│   시스템이 정지해 있고, 힘들의 합이 0                           │
│   ΣF = 0, Στ = 0                                                │
│                                                                 │
│   동적 평형 (Dynamic Equilibrium):                              │
│   정반응과 역반응이 같은 속도로 진행                            │
│   v_forward = v_backward                                        │
│                                                                 │
│   열역학적 평형 (Thermodynamic Equilibrium):                    │
│   엔트로피 최대, 자유에너지 최소                                │
│   dS = 0 (고립계), dG = 0 (등온등압)                           │
│                                                                 │
│   경제적 평형 (Economic Equilibrium):                           │
│   수요 = 공급, 가격 조정 압력 없음                              │
│   Q_d(P*) = Q_s(P*)                                             │
│                                                                 │
│   게임 이론적 평형 (Nash Equilibrium):                          │
│   어떤 플레이어도 일방적 이탈 유인 없음                         │
│   ∀i: u_i(s_i*, s_{-i}*) ≥ u_i(s_i, s_{-i}*)                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 왜 중요한가

- 시스템의 "자연 상태" 또는 "끌개" (attractor)
- 변화의 방향과 종착점 예측
- 안정성 분석의 기준점
- 최적화와 효율성의 기준

## 2. 분야별 발현

### 2.1 물리학/화학

```
화학 평형:
aA + bB ⇌ cC + dD

평형 상수: K = [C]^c[D]^d / [A]^a[B]^b

Le Chatelier 원리:
평형 시스템에 변화를 주면,
그 변화를 상쇄하는 방향으로 이동

열역학적 조건:
ΔG = ΔH - TΔS = 0 (평형에서)
```

### 2.2 경제학

```
시장 평형:
수요 곡선: Q_d = a - bP
공급 곡선: Q_s = c + dP
평형: P* = (a-c)/(b+d)

일반 균형 (Arrow-Debreu):
모든 시장이 동시에 청산
Walras 법칙: Σ p_i(D_i - S_i) = 0

Nash 균형 (게임):
각 플레이어의 전략이 상대방 전략에 대한 최적 반응
```

### 2.3 생태학

```
생태계 평형:
• 개체군 동역학의 안정점
• 포식자-피식자 균형 (Lotka-Volterra)
• 영양단계 간 에너지 흐름

항상성 (Homeostasis):
• 생리적 평형 유지
• 피드백 메커니즘
• 체온, pH, 혈당 조절
```

### 2.4 심리학/사회학

```
인지 평형 (Piaget):
• 동화와 조절의 균형
• 인지 발달의 동력

인지 부조화 (Festinger):
• 불일치 = 불평형 상태
• 평형 회복 동기

사회 평형 (Parsons):
• 사회 시스템의 안정 상태
• 기능적 균형
```

## 3. 안정성 분석

### 3.1 평형의 안정성 유형

```
┌─────────────────────────────────────────────────────────────────┐
│                    평형점 안정성 분류                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   안정 평형 (Stable):                                           │
│   작은 교란 후 원래 평형으로 복귀                               │
│       ╲    ╱                                                    │
│        ╲  ╱                                                     │
│         ●  ← 골짜기 바닥                                        │
│                                                                 │
│   불안정 평형 (Unstable):                                       │
│   작은 교란에도 평형에서 이탈                                   │
│         ●  ← 산꼭대기                                           │
│        ╱  ╲                                                     │
│       ╱    ╲                                                    │
│                                                                 │
│   준안정 평형 (Metastable):                                     │
│   작은 교란에는 안정, 큰 교란에는 불안정                        │
│            ╱╲                                                   │
│       ╲   ╱  ╲                                                  │
│        ● ╱    ╲  ← 국소 최소                                    │
│                 ● ← 전역 최소                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 선형 안정성 분석

```
dx/dt = f(x)

평형점 x*: f(x*) = 0

선형화: dx/dt ≈ J(x*) · (x - x*)

여기서 J = 야코비안 행렬

안정성 조건:
• 모든 고유값의 실수부 < 0 → 안정
• 어떤 고유값의 실수부 > 0 → 불안정
```

## 4. 핵심 통찰

### 4.1 평형 vs 정상상태

```
평형 (Equilibrium):
• 열역학적 구동력 = 0
• 순 흐름 = 0
• 자유에너지 최소

정상상태 (Steady State):
• 시간에 따른 변화 = 0
• 그러나 순 흐름 ≠ 0 가능
• 에너지 입력 필요 (비평형)

예: 생명체
• 열역학적 평형 아님 (죽음이 평형)
• 비평형 정상상태 유지
• 지속적 에너지 흐름 필요
```

### 4.2 다중 평형과 경로 의존성

```
다중 평형 시스템:
• 여러 안정 상태 공존 가능
• 초기 조건이 최종 상태 결정
• 이력현상 (Hysteresis)

함의:
• 역사가 중요 (path dependence)
• 갑작스러운 전환 가능 (tipping points)
• 되돌릴 수 없는 변화 (irreversibility)
```

## 6. 성찰적 질문

3. **사회 시스템**: 사회/경제 평형은 "자연스러운" 것인가, "바람직한" 것인가?
