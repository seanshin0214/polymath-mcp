### 1.2 복잡성의 정의 문제

```
복잡성이란 무엇인가?
━━━━━━━━━━━━━━━━━━━━━━━━

계산적 복잡도 (Kolmogorov):
• 최소 기술 길이
• 무작위 ≠ 복잡
• 압축 불가능성

조직적 복잡성:
• 구성요소의 수와 연결
• 계층적 구조
• 기능적 분화

논리적 깊이 (Bennett):
• 생성에 필요한 계산량
• 무작위는 단순 (즉시 생성 가능)
• 복잡한 것은 생성에 시간 필요

통합 정보 (Tononi):
• 전체가 부분보다 많은 정보
• 환원 불가능성
• Φ (파이) 측정

효과적 복잡성 (Gell-Mann):
• 규칙성의 기술 길이
• 완전 질서도, 완전 무질서도 단순
• 중간 지대가 복잡
```

## 2. 학제적 접근

### 2.1 물리학적 관점

```
엔트로피와 복잡성의 관계:
━━━━━━━━━━━━━━━━━━━━━━━━━━━

국소적 감소, 전역적 증가:
┌──────────────────────────────────────┐
│                                      │
│   ΔS_시스템 < 0 (질서 증가)         │
│   ΔS_환경 > 0 (무질서 증가)         │
│   ΔS_전체 = ΔS_시스템 + ΔS_환경 > 0 │
│                                      │
│   → 열역학 제2법칙 만족             │
│                                      │
└──────────────────────────────────────┘

비평형 열역학 (Prigogine):
• 산일구조 (Dissipative Structures)
• 에너지 흐름이 구조 생성
• 자기조직화의 조건

예시:
• Bénard 대류 셀
• 레이저
• BZ 화학반응 패턴
• 생명체

중력과 구조 형성:
━━━━━━━━━━━━━━━━━━━━━━━━━━━

중력의 역할:
• 음의 비열: 에너지 잃으면 뜨거워짐
• 국소적 붕괴 → 별 형성
• 균일 → 구조적

우주 구조 형성:
빅뱅 후 → 양자 요동 → 밀도 불균일 →
중력 붕괴 → 은하, 별, 행성
```

### 2.2 생물학적 관점

```
생명과 복잡성:
━━━━━━━━━━━━━━━━━━━━━━━━

슈뢰딩거의 질문 (1944):
"생명은 어떻게 엔트로피를 이기는가?"

대답: 이기지 않는다
• 생명 = 열린 시스템
• 음의 엔트로피 섭취 (negentropy)
• 환경 엔트로피 증가로 상쇄

진화와 복잡성:
━━━━━━━━━━━━━━━━━━━━━━━━

복잡성은 증가하는가?
• 논쟁적 주제
• 최대 복잡성은 증가
• 평균 복잡성은?

복잡성 증가의 메커니즘:
• 유전자 중복
• 내공생 (미토콘드리아)
• 다세포화
• 뇌의 확장

주요 전이 (Major Transitions, Maynard Smith):
원핵 → 진핵 → 다세포 → 사회성 →
언어 → 문명

정보 저장 방식의 변화:
유전자 → 후성유전 → 신경 → 문화 → 디지털
```

### 2.3 정보이론적 관점

```
정보와 복잡성:
━━━━━━━━━━━━━━━━━━━━━━━━

섀넌 정보:

• 불확실성의 측정
• 무작위가 최대 정보

의미 있는 정보:
• 섀넌 정보 ≠ 의미

• 기능적 정보 개념

물리적 정보:
━━━━━━━━━━━━━━━━━━━━━━━━

Wheeler: "It from Bit"
```

## 3. 핵심 메커니즘

```
자기조직화 조건:
━━━━━━━━━━━━━━━━━━━━━━━━

필요조건:
1. 많은 상호작용 요소
2. 비선형 상호작용
3. 에너지/물질 흐름 (비평형)
4. 피드백 루프

외부 설계 없이 내부적으로 구조 형성

━━━━━━━━━━━━━━━━━━━━━━━━

물리계:
• 레이저의 위상 동기화

• 결정 성장

화학계:
• BZ 반응의 시공간 패턴

생물계:
• 세포 분화
```

### 3.3 임계성 (Criticality)

```

━━━━━━━━━━━━━━━━━━━━━━━━━━━

"가장자리의 혼돈" (Edge of Chaos):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

최적 복잡성 영역:
• 너무 질서 → 경직

• 임계점 근처 → 계산 능력 최대

• 생명 시스템은 임계점 근처?
• 적응적 이점
• 진화적 선택?
```

## 6. 성찰적 질문

1. **정의의 문제**: 복잡성의 보편적 정의가 가능한가? 다양한 정의들은 같은 현상을 가리키는가?

2. **열역학적 역설**: 복잡성 증가가 엔트로피 증가와 양립가능하다면, 복잡성은 열역학적 필연인가?

3. **목적론의 유혹**: 복잡성이 증가하는 "이유"를 묻는 것은 정당한가? 목적 없는 과정에 방향이 있을 수 있는가?

4. **인간의 위치**: 인간 문명은 우주 복잡성의 정점인가, 아니면 하나의 사례인가?

5. **기술과 복잡성**: 인공지능은 복잡성의 새로운 형태인가? 생물학적 복잡성을 능가할 수 있는가?
